# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-25

### Added
- Initial release of MailO platform
- Full email server functionality (SMTP, IMAP)
- Modern React-based webmail interface
- Calendar with event scheduling and reminders
- Contact management system
- Task management with priorities
- Notes functionality
- Email campaigns feature
- Admin panel for user and domain management
- Two-factor authentication (2FA)
- Dark mode theme support
- Real-time notifications via WebSocket
- Email search functionality
- Send later (scheduled emails)
- Undo send feature
- Email snooze functionality
- Docker Compose deployment setup
- Comprehensive CI/CD automation with GitHub Actions
- Automated testing pipeline
- Security scanning with Trivy
- Dependabot integration for dependency updates
- Complete developer documentation
- API documentation
- Deployment guides

### Security
- JWT-based authentication
- bcrypt password hashing
- Rate limiting on authentication endpoints
- CORS protection
- Helmet.js security headers
- Input sanitization

### Infrastructure
- MongoDB for metadata storage
- Redis for caching and sessions
- MinIO for object storage (S3-compatible)
- BullMQ for background job processing
- Nginx for web serving
- Docker containerization
